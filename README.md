# create-tropa

Generator of `tropa` applications

# Quick Start

### Init App
```sh
npm init tropa hello-world
```
The `npm init` command will create `tropa` project at the `./hello-world` directory.

### Open project directory

```sh
cd ./hello-world    
```

### Install dependency
```sh
npm install
```

### Debug app
```sh
npm run debug
```

### Compile app
```sh
npm run build
```

### Start compiled app
```sh
npm run start
```

### Perform code analyse
```sh
npm run lint
```
